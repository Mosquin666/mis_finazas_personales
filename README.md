# 💼 Mis Finanzas Personales

Una app web progresiva (PWA) para llevar el control personal de ingresos, gastos y patrimonio. Minimalista, sin registro, sin servidor — los datos se guardan directamente en tu dispositivo.

---

## ✨ Características

- **Saldos iniciales patrimoniales** — registra lo que ya tenías en Disponible, Bancos e Inversiones sin que cuente como ingreso
- **Ingresos y gastos** — cada movimiento se asigna a una cuenta específica y una categoría
- **Subcuentas** — distingue entre Nu, BBVA, CETES, o lo que uses dentro de cada cuenta
- **Edición de movimientos** — porque errar es humano
- **Reporte mensual** — desglose por categoría, saldo por cuenta con detalle de subcuentas, y recomendaciones automáticas
- **Sin internet** — una vez instalada funciona completamente offline
- **Sin base de datos** — los datos viven en el localStorage del navegador, en tu propio dispositivo
- **Modo oscuro** — se adapta automáticamente a la preferencia del sistema

---

## 🚀 Uso

Puedes usarla directamente desde el navegador:

👉 **[Abrir app](https://Mosquin666.github.io/mis_finazas_personales/)**

O instalarla como PWA en tu celular para que se abra como app independiente:

**Android (Chrome):** Menú ⋮ → Agregar a pantalla de inicio

**iPhone (Safari):** Botón compartir → Agregar a pantalla de inicio

## 🗂 Categorías incluidas (Fue creada para uso personal en un entorno rural)
🏦 Bancos · 🚌 Transporte · 🍔 Alimentación · 🏠 Rancho · 📺 Internet y diversión · 👷 Construcción · 👨‍🌾 Café y plátano · 💳 Deudas · 🪙 Inversiones · 💰 Ahorro · 📦 Otros gastos

## 🛠 Tecnologías

Construida con HTML, CSS y JavaScript puro — sin frameworks, sin dependencias externas, sin build tools. Un solo archivo `index.html` que funciona en cualquier navegador moderno.

Los datos se persisten con `localStorage` del navegador.
